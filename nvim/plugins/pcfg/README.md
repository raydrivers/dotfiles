# `pcfg.nvim`
> So you can customize neovim per-project.

All `*.lua` scripts inside `<PROJECT_ROOT>/.pcfg` will be loaded as a part of your neovim config.
Feel free to define custom remaps, snippets here.

## Additional utilities

Provides build abstraction.

### Dependencies
- [`plenary.nvim`](https://github.com/nvim-lua/plenary.nvim)
